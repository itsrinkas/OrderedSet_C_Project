Detailed Description

Implementation of functions to manage an Ordered Set using a Double Linked List.

CE4703 Computer Software 3 Assignment 2
Data Type "Ordered Set" (Group Project)

This file contains functions to create, manipulate, and manage an ordered set of integers. The set is backed by a double linked list and includes operations to add, remove and perform set operations like union, intersection and difference.
